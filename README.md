# Emre Kitabevi Web Sitesi

Bu proje, HTML5 kullanılarak oluşturulmuş basit bir kitapçı web sitesidir.  
Site dört sayfadan oluşmaktadır: Ana Sayfa, Hakkında, Kitap Türleri ve İletişim.

## Sayfalar
- **index.html**
- **hakkinda.html**
- **servisler.html**
- **iletisim.html**

## Kullanılan Teknolojiler
- HTML5
